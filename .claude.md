# Garden Roots Frontend - AI Assistant Context

## Project Overview
React 19 e-commerce frontend for mango sales with Vite build system. Modern SPA with role-based access (customer/admin/delivery).

## Key Technologies
- **Framework**: React 19 with hooks
- **Build**: Vite 7.3.1
- **Styling**: CSS3 with custom components
- **State**: React Context (to be refactored)
- **HTTP**: Fetch API with custom wrapper

## Project Structure
```
src/
├── App.jsx                 # Root component
├── MangoStore.jsx         # Main app wrapper with role routing
├── context/AppContext.jsx # Global state (336 lines - needs refactoring)
├── components/            # All UI components
├── services/api.js        # API client (158 lines)
├── data/                  # Static fallback data
└── styles/               # CSS files
```

## Critical Issues (Priority Order)
1. **Security**: JWT tokens in localStorage → move to httpOnly cookies
2. **Architecture**: Monolithic AppContext (336 lines) → split into multiple contexts
3. **Performance**: No code splitting/lazy loading
4. **Accessibility**: Missing ARIA labels, keyboard navigation
